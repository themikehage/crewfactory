# Creación de Contenido y Copywriting (Content Creation)

Esta habilidad proporciona guías para redactar textos persuasivos, historias y documentación sobre el proceso de construcción de productos.

## Directrices para el Agente

### 1. Enfoque "Build in Public"
- Adopta una voz transparente y cercana: relata los desafíos superados, las decisiones difíciles tomadas y los logros técnicos.
- Mantén el lenguaje accesible: explica términos técnicos avanzados mediante analogías sencillas si el público es general.

### 2. Copywriting Persuasivo
- Escribe ganchos iniciales atractivos para captar la atención.
- Utiliza estructuras narrativas claras (p. ej., Problema → Agitación → Solución, o AIDA).
- Incluye llamados a la acción (CTA) claros al final del contenido para incentivar la participación o el uso del producto.
